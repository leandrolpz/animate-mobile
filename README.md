# AniMate Mobile 🎨📱

**AniMate Mobile** é uma ferramenta de animação 2D e desenho livre desenvolvida em **Kotlin** com **Jetpack Compose**. O projeto permite que usuários criem desenhos quadro a quadro, utilizando técnicas de animação como *Onion Skinning* (papel cebola) diretamente em seus dispositivos Android.

## ✨ Funcionalidades

- **Dashboard de Desenho (Studio):** Canvas interativo para criação de traços livres.
- **Sistema de Timeline:** Criação e navegação entre múltiplos frames para animação.
- **Onion Skinning:** Visualização do frame anterior em transparência para guiar o próximo movimento.
- **Ferramentas de Edição:** Pincel com ajuste de tamanho, borracha e ferramenta de desfazer (undo).
- **Interface Moderna:** UI/UX inspirada em ferramentas profissionais, utilizando Material 3 e Dark Mode.
- **Fluxo Completo de Usuário:** Telas de Home, Login, Cadastro e Recuperação de Senha integradas com Navigation Compose.

## 🚀 Tecnologias Utilizadas

- **Kotlin:** Linguagem principal do desenvolvimento Android moderno.
- **Jetpack Compose:** Toolkit moderno para construção de interfaces nativas.
- **Material 3:** Última versão do sistema de design da Google.
- **Navigation Compose:** Gerenciamento de rotas e navegação entre telas.
- **Canvas API:** Renderização gráfica de alta performance para os desenhos.

## 🛠️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/leandrolpz/animate-mobile.git
   ```
2. Abra o projeto no **Android Studio (Ladybug ou superior)**.
3. Certifique-se de que o **JDK 11 ou 17** esteja configurado.
4. Sincronize o Gradle.
5. Execute no Emulador ou dispositivo físico (API 31+ recomendado).
